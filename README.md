# Lyra-Lazuli
Lyra Lazuli Life: Gestor de juegos que sincroniza tus partidas locales en Google Drive (ideal para repacks/emuladores). Maximiza tus FPS con un optimizador extremo de sistema (libera RAM, Steam Lite, reduce input lag) e incluye una interfaz moderna estilo consola con descargas de metadatos automáticas.
